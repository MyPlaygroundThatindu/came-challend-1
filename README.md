# Hello World Program

### About the project
This is a simple Hello World program

### How to run the program
- Install java 17 in your computer
- Set up environmental variables for java 17
- Clone this progect to your computer
- Navigate to location wher you cloned Demo.java file
- Open command line interface from that location
- Run command "javac Demo.java"
- Then run command "java Demo"

[Official java documentation](https://docs.oracle.com/en/java/javase/17/)
